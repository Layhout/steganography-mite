# Digital Image Steganography

## Dependenices

- python-tk
- customtkinter
- pillow
- numpy
- pycryptodome
- matplotlib

## Requirement

Python version >= 3.12.2

## Installation

- Create python venv `python3 -m venv`
- Install all dependenices
- Run activate file to run env
- Run main.py `python main.py`
